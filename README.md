# text-editor-interface

A text editor interface with 2 modes: word,line. In line mode each line of text is numbered, a user can select one or many lines and save them to a new card object. In word mode, a user can select individual words and save to new card object.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/text-editor-interface.git
cd text-editor-interface
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
